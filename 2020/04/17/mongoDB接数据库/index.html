如何连mongoDB接数据库

  1. 下载mongoos第三方模块npm install mongoose 启动 ner start mongoDB
  2. 引入mongoose第三方模块，用来操作数据库
  3. 用connect方法来连接数据库，参数为mongodb://localhost/playground，最后一个是数据库名写上就会自动创建
  4. 分别用.then和.catch对数据库连接成功或失败一个反馈
const mongoose = require('mongoose')
mongoose.connect('mongoose://localhost/playground')
    .then(() => console.log('数据库连接成功'))
    .catch(() => console.log('数据库连接失败'))

如何创建一个集合以及向集合插入文档

  1. 创建集合规则，用new实例化mongoose，用Schema方法添加规则，规则为字段名和类型
  2. 创建集合，model方法第一个参数是集合名第二个参数是集合规则
  3. 创建文档，用new实例化集合，参数为一个对象每一项是具体的字段和值
  4. 将文档保存到数据库，文档名.save()
const personSchema = new mongoose.Schema({
    name: String,
    age: Number,
    hobbies: [String]
})
const Person = mongoose.model('Person', personSchema)
const person = new Person({
    name: 'zhangsan',
    age: 12,
    hobbies: ['sing', 'read']
})
person.save()

集合插入文档的另一种方式

之前我们插入文档是用new实例化集合，还有一个方式是用集合.create方法参数是一个对象，里面是具体的字段和值，可以用.then查看结果。

Person.create({ name: 'zhangsan', age: 12, hobbies: ['sing', 'read'] })
    .then(result => {console.log(result);})

find()查询指定集合的所有文档

Person.find().then(result => console.log(result))

find({条件})查询指定集合中满足条件的文档

Person.find({name:'李四'}).then(result=>console.log(result))

find({字段:{$gt:开始范围 , $lt: 结束范围}})查询字段位于之间的文档 &lt小于 &gt大于

Person.find({age:{$gt:20,$lt40}}).then(result=>console.log(result))

find({字段:{$in:包含项}})

Person.find({hobbies:{$in:['sing']}}).then(result=>console.log(result))

find().select('字段1，字段2..')查询指定字段，字段前加-，可以去除字段

Person.find().select('name,age,-id').then(result=>console.log(result))

find().sort('排序字段')默认按升序排序，加-按降序排序

Person.find().sort('-age').then(result=>console.log(result))

find().skip(跳过的条数).limit(限制显示结果的条数)

Person.find.skip(2).limit(3).then(result=>console.log(result))

findOne()返回指定集合第一个文档

Person.findOne().then(result=>console.log(result))

findOne({条件})返回指定集合满足条件的第一个文档

Person.findOne({age:14}).then(result=>console.log(result))

删除文档

查找一条文档并且删除，返回的是删除的文档，匹配多个文档会删除第一条

Person.findOneAndDelete({name:'张三'}).then(result => console.log(result))

删除多个文档，不加条件会删除全部文档

Person.deleteMany({}).then(result => console.log(result))

更新文档

更新一条文档

Person.updateOne({ name: '李四' }, { name: '李小四', age: 18 }).then(result => console.log(result))

更新全部文档

Person.updateMany({}, { age: 0 }).then(result => console.log(result))

集合关联

通常不同集合的数据之间是有关系的，例如文章信息和用户信息存储在不同集合中，但文章是某个用户发表的，要查询文章的所有信息包括发表用户，就需要用到集合关联。

  • 使用id对集合进行关联
  • 使用populate方法进行关联集合查询
const userSchema = new mongoose.Schema({
    name: {
        type: String,
        required: true
    }
});
const postSchema = new mongoose.Schema({
    title: String,
    author: {
        //类型为id，固定写法
        type: mongoose.Schema.Types.ObjectId,
        //关联集合user
        ref: 'User'
    }
})
const Post = mongoose.model('Post', postSchema);
const User = mongoose.model('User', userSchema)
Post.create({ title: '123', author: '5c0caae2c4e4081c28439791' }).then(result => console.log(result));
User.create({ name: '李四' }).then(result => console.log(result))
    //populate()关联集合查询
Post.find().populate('author').then(result => console.log(result))
